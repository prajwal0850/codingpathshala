Prototype-based Object-oriented<\/strong> – JavaScript is an object-oriented programming language. It used prototypes instead of classes. Objects are used to represent the real-world entity in the program.<\/p>\n\n\n\n

Interpreted Language<\/strong> – JavaScript is an interpreted programming language. It uses a just-in-time compilation technique at run-time.<\/p>\n\n\n\n

Event-Based Programming<\/strong> – JavaScript allows us to write codes that are executed differently under different events.<\/p>\n\n\n\n

Platform Independence<\/strong> – JavaScript codes are run on browsers. So irrespective of the machine, if the browser supports JavaScript, codes can run.<\/p>\n\n\n\n

\n\n\n\n

Why Learn JavaScript?<\/h2>\n\n\n\n
    \n
  • JavaScript is used on both the client-side and the server-side. On the client-side, the code is run and displayed by the browser. On the server-side, Node.js is used.<\/li>\n\n\n\n
  • JavaScript is a platform-independent language. Any browser supporting JavaScript can run the code irrespective of the operating system<\/li>\n\n\n\n
  • JavaScript uses the just-in-time compilation<\/strong> technique. Since the compilation is handled at run time, JavaScript is considered an interpreted language.<\/li>\n\n\n\n
  • JavaScript is a dynamically typed language. That is, a variable containing a number may be reassigned to a string.<\/li>\n<\/ul>\n\n\n\n
