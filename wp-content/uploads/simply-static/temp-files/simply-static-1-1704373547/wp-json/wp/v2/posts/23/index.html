What is Java?<\/h2>\n\n\n\n

Java is a popular programming language, created in 1995.<\/p>\n\n\n\n

It is owned by Oracle, and more than 3 billion<\/strong> devices run Java.<\/p>\n\n\n\n

It is used for:<\/p>\n\n\n\n

    \n
  • Mobile applications (specially Android apps)<\/li>\n\n\n\n
  • Desktop applications<\/li>\n\n\n\n
  • Web applications<\/li>\n\n\n\n
  • Web servers and application servers<\/li>\n\n\n\n
  • Games<\/li>\n\n\n\n
  • Database connection<\/li>\n\n\n\n
  • And much, much more!<\/li>\n<\/ul>\n\n\n\n

      \n
    • Java works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc.)<\/li>\n\n\n\n
    • It is one of the most popular programming language in the world<\/li>\n\n\n\n
    • It has a large demand in the current job market<\/li>\n\n\n\n
    • It is easy to learn and simple to use<\/li>\n\n\n\n
    • It is open-source and free<\/li>\n\n\n\n
    • It is secure, fast and powerful<\/li>\n\n\n\n
    • It has a huge community support (tens of millions of developers)<\/li>\n\n\n\n
